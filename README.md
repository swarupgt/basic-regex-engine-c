# basic-regex-engine-c
A basic regex engine written in C.

List of all operations supported:
 1. empty pattern
 2. exact match
 3. anchor ^
 4. anchor $
 5. metachar .
 6. closure *
 7. quantifier +
 8. non greedy ?
 9. character class [a-zA-Z0-9] along with individual characters
 10. metachars \d & \w

Currently does not support back referencing '()'
